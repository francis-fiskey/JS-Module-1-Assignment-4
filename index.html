Module 2 Assignment 2 Requirements

(Health application: BMI) Write a program that calculates a person's BMI.Use the following website to use the American Cancer Society indexes for BMI calculation.

Normal Weight Ranges: Body Mass Index (BMI)

For example, if a person is 5 feet and 10 inches, you will enter 5 for feet and 10 for inches.

Here is the pseudocode necessary to complete the code:

  • Input weight in pounds
  • Input height in feet
  • Input height in inches
  • Print BMI Calculation - Formula: weight (lb) / [height (in)]2 x 703
  • Print BMI Index determination

Workspace
